把注意力留给图片

选择一个能力，查看它在 VeloPic 工作台中的位置。

01

图库浏览

虚拟化网格按需加载缩略图，拖动滑块即可调整密度，大目录也保持顺畅。

从导入到整理

常用操作不绕路

↻

递归扫描

主动重新扫描目录，识别 JPG、PNG、WebP、HEIC、AVIF 等常见图片格式。

⌕

筛选与排序

按文件名、分辨率、类型筛选，并按时间、名称或大小进行持久化排序。

◎

沉浸查看

方向键和鼠标切换图片，滚轮缩放、拖拽平移，按 Esc 立即回到图库。

▣

收藏与相册

用收藏、相册、自定义分类和壁纸收藏夹组织图片，不移动原始文件。

⚡

离线智能分类

内置轻量 ONNX 模型，在本机辅助识别风景、城市、人物与自然图片。

▤

壁纸预览

先预览填充、适中或居中效果，确认后再设置为 Windows 桌面壁纸。

图片留在你的电脑里
